# よく使うコマンド
- 依存関係: `npm install`
- 開発サーバー: `npm run dev`
- ワーカー開発: `npm run dev:worker`
- 本番ビルド: `npm run build`
- 起動: `npm run start` / `npm run start:worker`
- Lint: `npm run lint`
- 型チェック: `npm run type-check`
- 単体テスト: `npm run test`
- E2E: `npm run test:e2e`
- Prisma生成: `npx prisma generate`
- Prismaマイグレーション(開発): `npx prisma migrate dev`
- Prisma Studio: `npx prisma studio`
