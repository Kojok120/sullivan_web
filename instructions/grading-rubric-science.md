## 理科の採点指針

教科名に「理科」を含む問題には、以下のルールを優先して適用してください。

### 化学式・イオン式の表記

- 添字の有無は同値: `H2O` と `H₂O`、`CO2` と `CO₂` はどちらも正解
- イオン式の符号の上付きの有無は同値: `Na+` と `Na⁺`、`Cl-` と `Cl⁻` は同じ
- 物質名と化学式のどちらで答えても、内容が一致していれば A（問題文の指示が無ければ）
- 慣用名と正式名の違い（例: `食塩` と `塩化ナトリウム`）は問題文脈で判定し、内容が一致していれば A〜B

### 用語のゆれ

- 漢字とひらがなの違いだけなら B 以上にしない（例: `光合成` と `こうごうせい`）
- 同義語は同等に扱う（例: `沸騰石` と `ふっとうせき`、`維管束` と `いかんそく`）
- 用語が惜しく概念は理解できている場合は C

### 単位と数値

- 理科では単位込みの解答が原則。単位を省略しているだけなら B
- 単位の表記ゆれ（`m/s` と `メートル毎秒`、`℃` と `度`）は同値
- 有効数字や小数点以下の桁数の指定がある場合はそれに従う

### 図・グラフからの読み取り

- 参考図版が与えられている問題では、必ず図を確認してから採点する
- 図から値を読み取る問題では、問題が想定する許容範囲内であれば A
- 図のラベル、矢印の向き、回路図の素子配置などは参考図版と照合する
- 作図問題は参考図版の正解例と要素（位置、向き、つなぎ方）が一致していれば A

### 部分点（C 評価）の目安

- 実験考察で結論は合っているが理由・根拠が不十分
- 化学反応式で反応物・生成物は合っているが係数が合っていない
- 観察結果の記述で要点の半分しか書けていない
