## 数学の採点指針

教科名に「数学」を含む問題には、以下のルールを優先して適用してください。

### 同値な表現は同じ正解として扱う

- 分数と小数: `1/2` と `0.5`、`3/4` と `0.75` は同値
- 約分の有無: 問題文に「最も簡単な分数で答えよ」等の指定がない限り、`4/8` と `1/2` はどちらも正解
- 文字式の項の順序: `2x + 3` と `3 + 2x` は同じ
- 文字式の係数: `1x` は `x`、`-1x` は `-x` として扱う
- 因数分解の順序: `(x+1)(x-2)` と `(x-2)(x+1)` は同じ
- 平方根: `2√3` と `√12`、`√4` と `2` は同値
- 連立方程式の解は順序問わず、すべての値が一致していれば正解

### 数値の許容誤差

- 円周率を `π` のまま残すか `3.14` で計算するかは、問題の指示に従う
- 指示がない場合、`π` を使った正答も `3.14` を使った正答もどちらも A
- 有効数字や四捨五入の指定がある場合はそれに従う

### 単位

- 単位の指定がある問題で単位が省略されているだけなら B（軽微なミス）
- 単位の表記ゆれ（`cm2` と `cm²`、`m/s` と `m/秒`）は同値

### 部分点（C 評価）の目安

- 計算過程は正しいが最終答えで符号ミスがある
- 連立方程式や 2 次方程式で片方の解だけ正しい
- 図形問題で求める対象（角度・面積・長さなど）を取り違えた
- 単位を含めて式は立てられているが、計算の途中でミスがある
